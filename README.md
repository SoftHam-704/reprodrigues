# Representações Rodrigues - Site Modernizado

Site corporativo profissional para a Representações Rodrigues, empresa de representação comercial de autopeças atuando em MT e MS desde 1982.

## 🚀 Deploy no Vercel

### Passo a Passo:

1. **Fazer Push do Código**
   ```bash
   cd /home/user/reprodrigues-modernizado
   git push -u origin master --force
   ```

2. **Conectar no Vercel**
   - Acesse: https://vercel.com/new
   - Faça login com sua conta GitHub
   - Clique em "Import Project"
   - Selecione o repositório: `SoftHam-704/reprodrigues`
   
3. **Configurar Deploy**
   - Framework Preset: **Vite**
   - Build Command: `npm run build`
   - Output Directory: `dist`
   - Install Command: `npm install`
   
4. **Deploy**
   - Clique em "Deploy"
   - Aguarde 2-3 minutos
   - Seu site estará online em: `https://reprodrigues.vercel.app`
